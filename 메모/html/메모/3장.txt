[이미지와 하이퍼링크]

<img>
웹 문서에 이미지를 삽입할 때 사용
=> <img src = "경로">

* img 태그의 속성
src 속성 : 이미지 파일 경로
alt 속성 : 이미지를 설명하는 대체 텍스트
width, height 속성 : 이미지 크기 조정

<a> ~ </a>
하이퍼링크

* a 태그의 속성
href : 링크한 문서나 사이트의 주소
target : 링크한 내용이 표시될 위치를 지정
download : 링크한 내용을 보여주는 것이 아니라 다운로드함
rel : 현재 무넛와 링크한 문서의 관계를 알려준다.
hreflang : 링크한 문서의 언어를 지정
type : 링크한 문서의 파일 유형 (ex. html, js, ...)

* target 속성
_blank : 새 창이나 새 탭에서 열기
_self : 링크가 있는 화면에서 열기
_parent : 링크 내용을 부모 프레임에 표시
_top : 프레임을 벗어나 링크 내용을 전체 화면에 표시

* 한 페이지 안에서 점프하기 - 앵커
1. 이동하고 싶은 위치에 id 속성을 이용하여 앵커 생성.
 (ex. <h2 id = "앵커이름"> )
2. 하이퍼링크에 지정한 앵커를 넣음.
 (ex. <a href = "#앵커이름"> )


* 이미지맵 - 한 이미지 상에 여러 다른 링크 만들기
<map name = "맵이름">
	<area shape = "rect", coords = "10, 10, 150, 200", ...>
	<area>
</map>
<img src = "이미지파일" usemap = "#맵이름">
