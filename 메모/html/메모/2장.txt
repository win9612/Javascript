[2장 텍스트 관련 태그들]

<hn> 태그 : 제목 표시
	각 웹 콘텐츠 영역에서 제목을 표시할 때 사용하는 태그
	h1 > h2 > ... > h6

<p> 태그 : 텍스트 단락
	입력한 내용 앞뒤로 빈 줄이 생기면서
	텍스트 단락이 만들어짐

<br> 태그 : 줄 바꾸기
	닫는 태그가 없음, 줄을 바꿀 위치에 삽입

<blockquote> 태그 : 인용문 넣기
	다른 텍스트보다 안으로 들여 써짐

<hr> 태그 : 분위기 전환
	주제가 바뀔 때 분위기 전환용
	수평 줄 생김

<pre> 태그 : 입력한 그대로 표시
	소스에 표시한 공백이 그대로 표시됨
	프로그램 소스 등을 표시할 때 사용

<strong> : 굵게 표시, 중요 내용
<b> : 단순히 굵게 표시

<em> : 강조하는 이탤릭체
<i> : 단순히 이탤릭체

<q> : 인용 내용 표시
	줄바꿈 없이 한 줄에 인용 내용 표시
	인용 내용 앞뒤에 따옴표가 추가됨

<mark> : 형광펜 효과

<span> : 줄 안에서 인라인 묶기
=> 특정 부분에 CSS를 적용하고 싶을 때 사용
(ex) <span style = "color : blue;">
<div> : 줄 바꿔 블록 단락으로 묶기


[목록 만드는 태그]
<ul>, <li> : 순서 없는 목록
	각 항목 앞에 불릿이 붙여짐
	CSS로 불릿 수정 가능

<ol>, <li> : 순서 목록
	각 항목 앞에 숫자가 붙여짐
<ol> 태그의 속성
	1. type : 불릿 앞의 숫자 조정
	2. start : 중간 번호부터 수정
	3. reserved : 역순 표시


<dl>
<dt>
<dd>
=> '제목'과 그에 대한 '설명'으로 이루어진 목록



======================================
[표를 만드는 태그]

<table> ~ </table>
표 전체 영역을 표시

<tr> ~ </tr>
행

<td> ~ </td>
셀

<th> ~ </th>
제목 셀

<td colspan = "합칠 열의 개수"> ~ </td>
<th colspan = "합칠 열의 개수"> ~ </td>
열 합치기 (가로가 길어짐!)

<td rowspan = "합칠 열의 개수"> ~ </td>
<th rowspan = "합칠 열의 개수"> ~ </td>
행 합치기 (세로가 길어짐!)

<caption> ~ </caption>
표 위에 제목 달기

<figure> ~ </figure>
각종 컨텐츠를 담는 컨테이너 역할을 하는 태그

<figcaption> ~ </figcaption>
figure 내의 태그, 이에 대한 설명하는 문구를 담는 태그

<thead> ~ </thead>
해당 영역을 표의 제목 부분으로 정의함

<tbody> ~ </tbody>
해당 영역을 표의 본문 부분으로 정의함

<tfoot> ~ </tfoot>
해당 영역을 표의 요약 부분으로 정의함

<col>
한 열에 있는 모든 셀에 같은 스타일을 적용하려고 할 때 사용
태그 안에 여러 속성 정의 가능

<colgroup> ~ </colgroup>
여러 열을 묶어 사용
영역 안에 해당 개수만큼 col 태그가 있어야 한다.
