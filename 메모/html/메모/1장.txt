[html/css 1강]

<!doctype html> 이 html문서의 맨 앞에 있어야 한다.

<html> 로 열고 </html>로 닫는다.

html에는 <head> 와 <body> 가 있다.

head는 웹 브라우저가 웹 문서를 해석하기 위해 필요한 정보들을 입력하는 부분

body는 실제로 웹 브라우저 화면에 나타날 내용
대부분 html태그들은 body 안에 들어감

<html lang = "ko"> :  문서에 쓰일 언어가 한국어
필수는 아니지만 사용 권장

* head 태그
<title> : 브라우저 제목 표시줄에 표시되는 내용
=> <title> 문서 제목 </title>
<meta> : 문자 인코딩 방법 및 문서 키워드와 요약 정보 지정
	우리가 보는 화면에서는 드러나지 않는,
	웹이랑 컴퓨터가 인식할 내용들?
=> <meta charset = "utf-8">

* body 태그
실제 브라우저에 표시될 내용들 입력
